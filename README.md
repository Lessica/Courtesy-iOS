## 描述
礼记是一个基于阿里公有云的腾讯互联轻应用。

## iOS 客户端环境部署
- 从 Mac App Store 安装 OS X with Xcode 7.3 (iOS SDK 9.0+)
- 安装 Cocoapods (https://cocoapods.org/)
```shell
$ sudo gem install cocoapods --pre
$ cd Courtesy-iOS
$ pod install --verbose
```
- 打开 Courtesy.xcworkspace 进行编译。

## Django 后端部署
- Python 2.7
- Django 1.9.2
- Nginx + Mysql
